# 🌿 Paradise Nursery

Paradise Nursery is a simple React-based plant shop application. It allows users to browse plants, add them to a shopping cart, update quantities, and remove items.

## 🚀 Features
- Landing page with company intro
- Product listing page
- Add to cart functionality
- Cart management (add/remove/update quantity)
- Redux state management

## 🛠️ Tech Stack
- React
- Redux Toolkit
- CSS

## 📦 Pages
- Home (Landing Page)
- Product List
- Cart Page
- About Us

```bash
npm install
npm start
